  • To Do List Application

    I was the leader for this application, I managed 5 people to create a to do list application for my Intro to Software Engineering class at Kennesaw State University. We focused on the documentation and development that went into creating applications for companies. From the design with the customer to the testing and development of the application. We used Xamarin with visual studio to create the GUI and the back end of the application. All was done in a 4-month period!

  • Database for a Movie Theater

    Me and 2 others created a database for a fake movie theater. We created Entity Relationship Diagrams, Data Dictionary Diagrams, CRUD Matrices, and the actual queries and tables for this fake movie theater. We used Microsoft Access to create the databases and the queries. The database would calculate how many tickets were sold and how many seats were available depending on what room the movie was in and how many seats were sold for that movie. The database also held basic information about the movie and times on the movies.
